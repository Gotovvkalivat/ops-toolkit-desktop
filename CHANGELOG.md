# Changelog

## 0.5.0

- Обновлён единый интерфейс расчёта и оформления без дублирования команд.
- Добавлена компактная панель реальных этапов оформления.
- Второстепенные действия перенесены в доступное меню `Ещё`.
- Сохранены отдельные клиенты расчёта и оформления, блокировки на время
  операций, серверный кеш и диагностика.
- Добавлена отдельная self-contained localhost-сборка для запуска в браузере
  без Node.js и установленного .NET.
- Добавлена общая команда сборки desktop и browser-дистрибутивов.
