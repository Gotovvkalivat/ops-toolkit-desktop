OPS Toolkit - версия для браузера
=================================

Как запустить
1. Откройте файл "Start OPS Toolkit.cmd".
2. Дождитесь автоматического открытия страницы в браузере.
3. Работайте с расчётом и оформлением так же, как в настольной версии.

Как остановить
Закройте окно OPS Toolkit Local Server. Пока оно открыто, локальный сервер работает.

Что устанавливать
Ничего. В папке уже находится .NET Runtime. Node.js, .NET SDK и другие программы
пользователю не нужны.

Важно
- Не удаляйте папку public и не переносите EXE отдельно от остальных файлов.
- Сервер доступен только на этом компьютере через 127.0.0.1.
- Если основной порт занят, приложение автоматически выберет следующий свободный.
